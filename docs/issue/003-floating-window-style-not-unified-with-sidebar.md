## 现象/背景

当前悬浮翻译窗与固定侧边栏在展示同一段译文时，视觉样式不一致。用户明确反馈差异主要集中在 LaTeX 公式样式，同时普通文本字体也有可见差别。

## 复现步骤

1. 准备一份包含行内与块级公式的 PDF（样本文档：`<TBD>`）。
2. 在固定侧边栏查看对应译文输出。
3. 打开悬浮翻译窗查看同一段译文（入口：`<TBD>`）。
4. 对比文本字体与 LaTeX 渲染样式。

- 需要用户补充：
  - 悬浮窗打开入口与对比路径
  - 最小复现样本文本（含公式片段）
  - 差异截图或具体参数（字体族/字号/行高/公式间距）

## 预期结果 / 实际结果

- 预期：悬浮翻译窗与固定侧边栏在字体和 LaTeX 渲染表现上保持一致。
- 实际：LaTeX 样式明显不一致，普通字体也存在差异。

## 根因分析

当前根因未确认，先记录假设与证据缺口：

- 假设 A：两种容器使用了不同的样式入口或 CSS 作用域，导致 typography token 不一致。
- 假设 B：两种容器的 markdown/数学渲染链配置不完全一致（如 KaTeX 参数差异）。
- 假设 C：悬浮窗字体回退链与固定侧边栏不同，导致中文与数学符号混排差异。

证据缺口：
